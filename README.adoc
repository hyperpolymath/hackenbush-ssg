= hackenbush-ssg
:toc: auto
:toclevels: 3

**The Conway's Game of Life static site generator.**

== Overview

hackenbush-ssg is an experimental static site generator that uses
Conway's Game of Life (a Turing-complete cellular automaton) to
generate content patterns and site structure.

== Language Enforcement

[IMPORTANT]
====
This SSG uses JavaScript to implement the Game of Life cellular automaton.
The SSG logic is the Life simulation itself.

The **ONLY** non-JS code allowed is the ReScript MCP adapter in `adapters/`.
====

== Features

* Turing-complete content generation via cellular automata
* Pattern-based page layouts (gliders, oscillators, etc.)
* Generative art integration
* Interactive life simulations embedded in pages
* Educational demonstration of computational universality

== Requirements

* Node.js 18+ or Deno

== Building

[source,bash]
----
node src/main.js build
----
